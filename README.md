# ⚖️ BMI Calculator (Bilingual)

A sleek, highly customizable, and bilingual (English/Farsi) Body Mass Index (BMI) calculator built with Flutter. This application provides real-time BMI calculations, health analysis based on age and gender, and a dynamic user interface.

## ✨ Features

| Feature | Description |
| :--- | :--- |
| **🌍 Bilingual Support** | Seamless real-time switching between English and Farsi (Persian). |
| **💾 Persistent Storage** | Automatically saves the user's last BMI calculation using `SharedPreferences`. |
| **🎨 Dynamic Typography** | Smart font-switching based on the active language (BNazanin/BTitr for FA, ETimes for EN). |
| **🎯 Advanced Logic** | Tailored health advice adjusting for age (e.g., specific ranges for adults over 60) and gender. |
| **📱 Custom UI Components** | Includes custom-built continuous sliders, interactive increment/decrement cards, and SVG iconography. |

## 🚀 Built With

*   **Framework:** [Flutter](https://flutter.dev/)
*   **Language:** Dart
*   **Storage:** `shared_preferences`
*   **Assets:** Custom TTF Fonts & `flutter_svg`
